# Mahzen Bot

Mahzen Bot is a social media downloader designed to help users easily download content from various social media platforms.

## Features
- Download videos, images, and stories.
- Supports multiple platforms including Twitter, Instagram, and Facebook.
- User-friendly interface.
- Fast and reliable downloading.

## Installation
1. Clone the repository.
2. Install dependencies using `pip install -r requirements.txt`.
3. Run the bot with `python main.py`.

## Usage
Just provide the URL of the content you want to download and the bot will take care of the rest!
